# Ada LK News Blogger Automation

This script automatically scrapes news from Ada LK and posts them to both a GitHub README.md file and a Blogger blog.

## Setup Instructions

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Set up Google Blogger API:
   - Go to the [Google Cloud Console](https://console.cloud.google.com/)
   - Create a new project
   - Enable the Blogger API v3
   - Create OAuth 2.0 credentials
   - Download the credentials and save them as `credentials.json` in the project directory

3. Create a `.env` file with your Blogger blog ID:
```
BLOG_ID=your_blog_id_here
```

4. First run authentication:
   - Run the script for the first time
   - A browser window will open asking you to authorize the application
   - Grant the necessary permissions
   - The token will be saved for future use

## Usage

Simply run the script:
```bash
python blogger_news_poster.py
```

The script will:
1. Scrape new articles from Ada LK
2. Update the README.md file with the new articles
3. Post the new articles to your Blogger blog
4. Keep track of posted articles to avoid duplicates

## Features

- Automatic news scraping from Ada LK
- Posts to both GitHub README and Blogger
- Handles images and formatting
- Maintains a log of posted articles
- Preserves static content in README.md

## ත්‍රීපෝෂ සමාගමට නව සභාපතිවරයෙක්

ශ්‍රී ලංකා ත්‍රීපෝෂ සමාගමේ නව සභාපතිවරයා ලෙස දේශීය හා ජාත්‍යන්තර ව්‍යාපාරික ක්ෂේත්‍රයේ ඉහළ පළපුරුද්ද සහ අත්දැකීම් සහිත අමල් නිරෝෂණ අත්තනායක මහතා පත් කිරීමට සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍යාංශය පියවර ගෙන ඇත.

ඊට අදාල පත්වීම් ලිපිය  සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍ය වෛද්‍ය නලින්ද ජයතිස්ස මහතා විසින් අමල් නිරෝෂණ අත්තනායක මහතා වෙත ලබාදීම අද (13) සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍යාංශයේ දී සිදු විය.

වසර 25 කට අධික කාලයක් පුරා දේශීය හා ජාත්‍යන්තර ව්‍යාපාරික ක්ෂේත්‍රයේ ඉහළ පළපුරුදක් සහිත අමල් නිරෝෂණ අත්තනායක මහතා මහනුවර ධර්මරාජ විද්‍යාලයේ ආදි ශිෂ්‍යයෙකි. පාසල් වියේ  දී ජනාධිපති බාල දක්ෂයෙකු ලෙස කටයුතු කර ඇති ඒ මහතා ශ්‍රී ජයවර්ධනපුර විශ්ව විද්‍යාලයෙන් ශාස්ත්‍රවේදී ගෞරව උපාධිධාරියෙකි. විශ්ව විද්‍යාල ක්‍රීඩා සංගමයේ ලේකම්වරයා ලෙස ද කටයුතු කර ඇති ඒ මහතා ටෙනිස් ක්‍රීඩාවෙන් මෙරට නියෝජන්‍ය කරමින් ජාත්‍යන්තරය තරඟාවලි සඳහා ද සහභාගී වී ඇත.

ලක්විජය බලාගාරයෙ පලමු, දෙවන හා තුන්වන අධියර තුනක් ඇති අතර එයින් තුන්වන ජනන යන්ත්‍රය මෙලෙස ජනන කටයුතු නවතා වාර්ෂීකව සිදුකරන නඩත්තුව සිදු කිරීමට නියමිත බව කියයි.

රුපියල් පන් කෝටි පනස් ලක්ෂයක් වටිනා විස්කි බෝත්ල් 673 ක් සහ ඉන්දියාවේ නිෂ්පාදිත කරදමුංගු කිලෝ ග්‍රෑම් 75 ක් , කටුනායක ගුවන් තොටුපොළ පළමු මහළේ , තීරු බදු රහිත සාප්පු සංකීර්ණ පරිශ්‍රයේ අතහැර දමා ගොස් තිබියදී 06/13 දින අලුයම ගුවන් තොටුපොළ රේගු නිලධාරීන් පිරිසක් විසින් සොයා ගෙන අත් අඩංගුවට ගෙන තිබේ .

මෙම විස්කි සහ කරදමුංගු තොගය කුමන ගුවන් යානයක පැමිණි කුමන ගුවන් මගියෙකු විසින් කටුනායක ගුවන් තොටුපොළ වෙත රැගෙන එන්නට ඇත්දැයි සොයා රේගු නිලධාරීන් විමර්ශන කටයුතු ආරම්භ කර තිබේ .

මෙහිදී මෙම විස්කි සහ කරදමුංගු තොගය දමා ගොස් තිබූ ගුවන් තොටුපොළ තීරු බදු රහිත සාප්පු සංකීර්ණයේ , සාප්පු වල සේවකයින්ගෙන් ප්‍රශ්ණ කරමින්  සහ ගුවන් තොටුපොළ ආරක්ෂක කැමරා පද්ධතිය නිරීක්ෂණය කරමින් මේවායේ හිමිකරුවන් අනාවරණය කර ගැනීමට රේගු නිලධාරීන් ප්‍රයත්න දරමින් සිටිති .

## ඊශ්‍රායලය ඉරානයට දැවැන්ත ප්‍රහාරයක් එල්ල කරයි

Published on June 13, 2025, 11:31 AM

ඉරානයේ න්‍යෂ්ටික වැඩසටහනේ "හදවත" ඉලක්ක කරගනිමින් ඉරානය වෙත දැවැන්ත ප්‍රහාර මාලාවක් ඊශ්‍රායලය විසින් එල්ල කර තිබේ. ඉරාන රාජ්‍ය මාධ්‍යවලට අනුව, මෙම ප්‍රහාරවලින් ඉරානයේ ඉස්ලාමීය විප්ලවවාදී ආරක්ෂක බලකායේ ප්‍රධානී හොසේන් සලාමි මෙන්ම න්‍යෂ්ටික විද්‍යාඥයින් පිරිසක්ද ද මිය ගොස් ඇති බවට තහවුරු කර තිබේ.

ඊශ්‍රායල ජනාධිපති බෙන්ජමින් නෙතන්යාහු පැවසුවේ මෙම ප්‍රහාර "ඊශ්‍රායලයේ පැවැත්මට ඉරාන තර්ජනය මැඩපැවැත්වීම සඳහා ඉලක්ක කරගත් මිලිටරි මෙහෙයුමක්" බවයි.

ප්‍රතිප්‍රහාර එල්ල විය හැකි බැවින් ඊශ්‍රායලයේ හදිසි තත්වයක් ප්‍රකාශයට පත් කර තිබේ.

මෙම ප්‍රහාර සඳහා එක්සත් ජනපදය  සම්බන්ධ නොවූ බවත් කිසිදු සහායක් ලබා නොදුන් බවත් එක්සත් ජනපද රාජ්‍ය ලේකම් මාර්කෝ රුබියෝ පවසා තිබේ.

ඉරානයේ සන්නද්ධ හමුදාවන්ගේ ප්‍රකාශකයෙකු පවසා ඇත්තේ එක්සත් ජනපදය සහ ඊශ්‍රායලය යන දෙකම ප්‍රහාර සඳහා "දැඩි මිලක්" ගෙවනු ඇති බවයි.

## කළුතර ප්‍රාදේශීය සභාවේ විදුලි සේවකයා ඝාතනය කර ඇත්තේ අත් වැරදීමකින්

Published on June 13, 2025, 10:58 AM

පසුගිය  7 දා අළුයම සිය නිවසේදී කළුතර සේවකයකු කපාකොටා ඝාතනය කර ඇත්තේ අත් වැරදීමකින් බවට අත්අඩංගුවට ගත් ප්‍රධාන සැකකරුගෙන් හෙලිව ඇති බව කළුතර කොට්ඨාශ අපරාධ විමර්ශන ඒකකය පවසයි.

කළුතර ප්‍රාදේශීය සභාවේ විදුලි සේවක, දෙදරු පියෙකු වූ සුගත් ප්‍රසංග මහතා පදිංචිව සිටී කළුතර පනාපිටියේ පිහිටි නිවසට එදින අළුයම තියුණු ආයුධ රැගෙන කඩාවැදුන පිරිසක් දොර කඩා දරු දෙදෙනා ඉදිරිපිටදී කපාකොටා ඒ මහතාව ඝාතනය කර බිරිඳට කඩු ප්‍රහාරයක් එල්ලකර පලාගොස් තිබුණි.

ඒ අනුව පරීක්ෂණ කල කළුතර කොට්ඨාස අපරාධ විමර්ශන ඒකකයේ උප පොලිස් පරීක්ෂක පී ආර් වීරසේන මහතා ඇතුළු නිළධාරීන්  ප්‍රදේශයේ සීසීටීවී දර්ශණ පරීක්ෂා කිරීමෙන් පසු කළුතර මෙනේරිතැන්න ප්‍රදේශයේ පදිංචි ප්‍රධාන සැකකරු අත්අඩංගුවට ගෙන තිබේ.

ඝාතනයට යොදාගත් තියුණු ආයුධ දෙකක් සහ ඝාතනය සැලසුම් කිරීම ට යොදාගත් නවීන ජංගම දුරකථනයක් සහ ඝාතනයට පැමිණි ත්‍රීරෝද රථය ද පොලිස් භරයට ගත්බව පොලිසිය කීය.

ඝාතනයට ලක්වූ පුද්ගලයාගේ නිවස ආසන්නේ පදිංචිව සිටින  මත්ද්‍රව්‍ය ජාවාරමකට සම්බන්ධ වෙනත් පුද්ගලයකු ඝාතනය කිරීමට ලද කොන්ත්‍රාත්තුවක් මත සැකකරුවන් පැමිණ ඇති අතර, එහිදී අධිකලෙස බීමත්ව සිටී පුද්ගලයකු  ඝාතනයට ලක්වූ ප්‍රාදේශීය සභාවේ සේවකයාගේ නිවස පෙන්වීම නිසා සිදුවූ අත්වැරදීමකින් ඝාතනය සිදුව ඇති බවට ප්‍රධාන සැකකරුගේ ප්‍රකාශ වලින් හෙලිව ඇති බව පොලිසිය කියයි.

සැකකරු  කළුතර මහේස්ත්‍රාත් අධිකරණයට ඉදිරිපත් කිරීමට නියමිතව තිබුණි .

## අනුරපුරේ ආපු කට්ටිය කළා වැවත් කාලා

Published on June 13, 2025, 09:51 AM

පොසොන් වන්දනාවේ අනුරාධපුරේ පැමිණි බැතිමතුන් විනාශ කර දමා තිබු කළාවැව අවට පිරිසිදු කිරීම රාජ්‍ය නිලධාරීන් රැසකගේ සහභාගිත්වයෙන් 12 දින දවස පුරා පැවැත්විණි.

ශ්‍රී ලංකා මහවැලි  අධිකාරියේ බෝතැන්න ජලාශ භාර ප්‍රධාන ඉංජිනේරු බුද්ධික මහේෂ් නවරත්න මහතාගේ ප්‍රධානත්වයෙන් මහවැලි අධිකාරියෙ ගල්නෑව සහ ඉපලෝගම ප්‍රාදේශීය සභා දේශීය ලේකම් කාර්යාල ඇතුළු රාජ්‍ය නිලධාරීන්ගේ සහභාගිත්වයෙන් මෙම ශ්‍රම දානය පැවැත්විණි.

අනුරාධපුර වන්දනාවේ පැමිණි බොහෝ දෙනෙකුට කිලෝ මීටර් හතකින් දුර වූ අලංකාර මනස්කාන්ත  වැව් බැම්ම දැඩි ලෙස අපවිත්‍ර කොට තිබිණි.

දන්සැල් වලින් රැගෙන ආ ආහාරපාන ලබා ගෙන දිරන සහ දිරන ද්‍රව්‍ය බෝතල් මෙන්ම මත්පැන් බෝතල් ද රැසක් මෙම වැව් බැම්මේ තැන තැන දමා තිබිණි ඊට අමතරව වැව් ජලයට ද අප ද්‍රව්‍ය එක්කර තිබීම විශේෂත්වයකිි.

රාජ්‍ය නිලධාරීන් කණ්ඩායම්වලට බෙදී කිලෝමීටර් හතක් වූ වැව් බැම්ම සම්පූර්ණයෙන්ම පිරිසිදු කිරීම සිදුකළ බවත් ටොන් ගනාවක් අපද්‍රව්‍ය එකතුකළ බවත් ප්‍රධාන ඉංජිනේරුවෝ බුද්ධික මහේෂ් නවරත්න මහතා සඳහන් කළේය

පසුගිය සිංහල අලුත් අවුරුදු සමයේත් වෙසක් සැමරුම් සමයේත් පොසොන් සැමරුම් සමයේත් මේ ආකාරයටම කලා වැව ප්‍රදේශයේ  ජනතාව විසින් පරිසරය විනාශ කොට තිබිණ .

## අදත් මිලි මීටර් 100ක් දක්වා වැසි

Published on June 13, 2025, 08:19 AM

බස්නාහිර, සබරගමුව, මධ්‍යම, දකුණු හා වයඹ පළාත්වල විටින් විට වැසි ඇතිවන බව කාලගුණ විද්‍යා දෙපාර්තමේන්තුව පවසයි.

එම දෙපාර්තමේන්තුව අද පෙරවරුවේ නිකුත් කළ නිවේදනයක දැක්වෙන්නේ බස්නාහිර හා සබරගමුව පළාත්වලත් නුවරඑළිය, මහනුවර, ගාල්ල මෙන්ම මාතර දිස්ික්කවලත් ඇතැම්ස්ථානවලට මි.මී. 100 කා පමණ තද වැසි ඇතිවිය හැකි බවයි.

උතුරු පළාතෙත් අනුරාධපුර දිස්ත්‍රික්කයේත් වැසි වාර කිහිපයක් ඇතිවිය හැකි යැයි එම දෙපාර්තමේන්තුව දන්වා සිටියි.

මධ්‍යම කඳුකරහේ බටහිර බැවුම් ප්‍ර දේශවලත් උතුරු, උතුරු-මැද, බස්නාහිර, සබරගමුව, වයඹ හා දකුණු පළාත්වලත්  ත්‍රිකුණාමලය දිස්ත්‍රික්කයේත් විටින් විට හමන පැ.කි.මී. (50-60) ක පමණ තද සුළංඇතිවිය හැකි යැයි කාලගුණ විද්‍යා දෙපාර්තමේන්තුව වැඩිදුරටත් පවසයි.

## බෝරා සමුළුවට පහසුකම් සැලසීමට රජය මැදිහත් වෙයි

Published on June 13, 2025, 07:27 AM

ශ්‍රී ලංකාවේ පැවැත්වීමට නියමිත ගෝලීය බෝරා ප්‍රජාවගේ අධ්‍යාත්මික සමුළුව සහ ප්‍රදර්ශනය සාර්ථක කර ගැනීමට අවශ්‍ය උපරිම සහාය ලබා දෙන ලෙස ජනාධිපති ලේකම් ආචාර්ය නන්දික සනත් කුමානායක මහතා අදාළ නිලධාරීන්ට උපදෙස් ලබා දී ඇත.

ජනාධිපති ලේකම්වරයා විසින් මෙම උපදෙස් ලබා දෙනු ලැබුවේ, ගෝලීය බෝරා සමුළුව සහ ප්‍රදර්ශනය පිළිබඳ ජනාධිපති කාර්යාලයේදී ඊයේ (12) පැවති සාකච්ඡාවේදීය.

බෝරා ප්‍රජාවගේ අධ්‍යාත්මික සමුළුවට සමගාමීව ජූනි මස 27 වන දා සිට ජූලි 05 වන දා දක්වා බෝරා ප්‍රජාවේ අධ්‍යාත්මික නායක ආචාර්ය අතිඋතුම් සෙයියදිනා මුෆද්දල් සෙයිෆුද්දීන් සහෙබ්තුමාගේ සහභාගීත්වයෙන් බම්බලපිටිය බෝරා දේවස්ථානය සහ ශ්‍රී ලංකා ප්‍රදර්ශන හා සම්මේලන මධ්‍යස්ථානය කේන්ද්‍ර කර ගනිමින් මෙම බෝරා සමුළුව හා ප්‍රදර්ශනය පැවැත්වේ.



මෙම ප්‍රදර්ශනය සහ සමුළුව සඳහා ඉන්දියාව ඇතුළු රටවල් රැසකින් බෝරා ප්‍රජාවේ සාමාජිකයන් පහළොස් දහසක් පමණ මෙරටට පැමිණීමට නියමිත අතර ඔවුන් වෙනුවෙන් ගුවන් තොටුපොළ පහසුකම් ඇතුළු සියලු යටිතල පහසුකම් සැපයීම පිළිබඳව මෙහිදී දීර්ඝ ලෙස සාකච්ඡා කර ඇත.

විදේශ කටයුතු අමාත්‍යාංශයේ ලේකම් අරුණි රණරාජා, ජනාධිපති ජ්‍යෙෂ්ඨ අතිරේක ලේකම් රෝෂන් ගමගේ ‍යන මහත්ම මහත්මීන් ඇතුළු බෝරා ප්‍රජාවේ ආගමික නායකයෝ පිරිසක්ද මෙම අවස්ථාවට එක්ව සිටියහ.

## ගුවන් අනතුරින් 241ක් ජීවිතක්ෂයට, මිය ගිය පුද්ගලයෙකුට ඉන්දීය රු. කෝටියක වන්දියක්

Published on June 13, 2025, 07:20 AM

එයාර් ඉන්දියා ගුවන් අනතුරින් මිය ගිය සංඛ්‍යාව 241ක් බවට ගුවන් සමාගම තහවුරු කරයි.එසේම මෙම අනතුරින් එක් පුද්ගලයෙක් දිවි බේරාගෙන තිබේ. මියගිය අයගේ පවුල් වෙනුවෙන් මිය ගිය එක් පුද්ගලයෙක් වෙනුවෙන් ඉන්දීය රුපියල් කෝටිය බැගින් වන්දි ගෙවන බව එයා ඉන්දියා ගුවන් සේවය හිමි ටාටා සමාගම පවසයි.

## පාසලේ පංති කාමරයට ගසක් කඩා වැටී සිසුවෙක් මරුට

Published on June 12, 2025, 09:01 PM

බලන්ගොඩ රජවක විද්‍යාලයේ අද (12දා) සවස ගසක් කැඩා වැටීම් හේතුවෙන් පාසල් සිසුවකු ජීවිතක්ෂයට පත් වූ බව බළන්ගොඩ පොලිසිය පැවසීය.

දොළහ වසර පන්ති කාමරයකට මෙම ගසක් කඩා වැටී ඇති අතර ළමුන් දාහත් දෙනෙක් බලන්ගොඩ මූලික රෝහල්ට ඇතුලත් කර තිබේ. වැඩිදුර පරීක්ෂණ බලංගොඩ පොලිසිය සිදු කරයි.

## මහනුවර මහ නගර සභාව තුළ පක්ෂ විපක්ෂ කියලා දෙකොටසක් නෑ

Published on June 12, 2025, 08:57 PM

ජාතික ජන බලවේගයට මෙවර බලය හිමිවූ මහනුවර මහ නගර සභාවේ මංගල මහා සභා රැස්වීම එහි පක්ෂ විපක්ෂ මන්ත්‍රීවරුන් 46 දෙනාගේ සහභාගිත්වයෙන් එහි ප්‍රධාන සභා ශාලාවේදී පැවැත්විණි.

මහනුවර මහ නගර සභාවේ 16 වැනි පුරපතිවරයා වශයෙන් තේරී පත්වු චන්ද්‍රසිරි විජේනායක මහතාගේ ප්‍රධානත්වයෙන් රැස්වීම  ආරම්භ වූ අතර එහිදී මෙවර නගර සභාවේ බහුතරයක් වූ නවක මන්ත්‍රීවරුන් හඳුන්වා ගැනීම සිදු විය.

එහිදී බහුතර මන්ත්‍රීවරුන්ගේ අදහස වූයේ මහනුවර මහ නගර සභාවේ පක්ෂ විපක්ෂ යනුවෙන් දෙකොට්ඨාසයක් නොසිටිය යුතු බවත් නගරයේ සුභ සිද්ධිය සහ සංවර්ධනය වෙනුවෙන් සියලු මන්ත්‍රීවරුන් එකා වැඩ කටයුතු කළ යුතු බවයි.

එහිදී නියෝජ්‍ය නගරාධපති රුවන් කුමාර ගලපිටිය මහතා කතා කරමින් අවධාරණය කර සිටියේ මහනුවර නාගරික බල ප්‍රදේශයේ ඇතැම් ප්‍රදේශවලට සභාවේ මුදල් වැය කොට සංවර්ධන කටයුතු සිදුකොට ඇතත් ඇතැම් ස්ථානවල ජනතාවට මූලික පහසුකම් හෝ නොමැති බවයි. සභාවේ දූෂණය  වංචාව අඩු වැඩි වශයෙන් තිබෙන බවත් අඩුම වශයෙන් මහජනතාවගේ බදු මුදල් නාස්තිය හෝ අවම කිරීමට පියවර ගත යුතු තිබෙන බවද පැවසීය.

මහනුවර මහ නගර ස්වභාව තුළ ආණ්ඩු පක්ෂය හෝ විපක්ෂය කියා කොටස් දෙකක් නොමැති බවත් සිටින්නේ නගරාධිපතිවරයා සහ සභිකයන් පමණක් බවද අවධාරණය කළේය.

සභාවේ ජ්‍යෙෂ්ඨ ජාතික ජනබල වේගය මන්ත්‍රී මොහොමඩ් මුතාලිබ් මහතා කරුණු දක්වමින් පවසා සිටියේ මහනුවර මානගල සභාව තුළ ඉදිරියේදී සිදුකිරීමට විශේෂ කර්තව්‍ය රැසක් ඇති බවත් ජනතාව විශාල බලාපොරොත්තුවක් ඇතුව ජාතික ජන බලවේගයට මෙම නගර සභාවේ බලය හිමිකර දී ඇති බවද එහි විශ්වාසය පළුදු නොවන ආකාරයට වැඩ කටයුතු කළ යුතුව තිබෙන බවද පැවසීය.

පසුගිය අබාවාරයේදී ජනතා විමුක්ති පෙරමුණේ තමන් ඇතුළු සභිකයන් දෙදෙනකු පමණක් සිටිමින් නගර සභාවේ දූෂණය වංචාව පිටුදැකීම සඳහා විශාල මෙහෙවරක් සිදුකළ බවත් එය ඉදිරියටත් කරගෙන යාමට ජනතාව තවත් විශාල වගකීමක් පවරා ඇති බවද අවධාරණය කළේය.

මෙරට දේශපාලන සංස්කෘතිය වෙනස් කිරීම සඳහා මෙම බලය ජනතා විසින් ලබාදී ඇති බවත් ජනතාවට තවත් සමීප වී තවත් ඉදිරියට ගෙන යා යුතුව තිබෙන බවද නගර සභාව තුළ වංචා දූෂණ රැසක් සිදුවෙමින් තිබෙන බවට ජනතාව අතර මතයක් තිබෙන බැවින් එ් පිළිබඳව විශේෂ අවධානය යොමු කොට කටයුතු කළ යුතුව තිබෙන බවද පැවසීය.

පසුගිය සභා වාරයේ මන්ත්‍රීවරුන් වශයෙන් සිටියදී තමන් ඇතුළු වත්මන් නගරාධිපතිවරයා විසින් සභාව තුළ සිදුවූ රුපියල් කෝටි පහකට අධික ජල බට වංචාවක් අල්ලස් හා දූෂණ විමර්ශන ඒකකයට යොමු කළ බවත් මෙම නගර සභා ඉතිහාසයේ විශාලතම හොරකම එයද මුතාලිබ් මහතා වැඩිදුරටත් අවධාරණය කළේය.

## ඊශ්‍රායෙලයේ සිටින ලාංකිකයන්ට තම ආරක්ෂාව සලසා ගන්නැයි දැනුම් දෙයි

Published on June 13, 2025, 04:55 PM

ඊශ්‍රායලය හා ඉරානය ආශ්‍රිත කලාපයේ ඇති තිඛෙන නොසන්සුන්තාවයක් හේතුවෙන් ඊශ්‍රායලය, ඉරානය ඇතුළු කලාපය පුරා ගුවන් කලාප හරහා ගුවන් ගමන් නතර කොට ඇති අතර ගුවන් ගමන් මාර්ග වෙනස් කොට යුරෝපය හා ආසියාව අතර ගමන් වාර ආරම්භ කොට තිබෙන බවත්, මේ හේතුවෙන් බොහෝ ගුවන් ගමන්වල කාලය දිගුවිය හැකිබවත්, ඊශ්‍රායලයේ ශ්‍රී ලංකා තානාපති විසින් පණිවුඩයක් නිකුත් කරමින් සඳහන් කර තිබේ.

ඊශ්‍රායලයට පැමිණීමට අපේක්ශා කරන පිරිස් තම පැමිණීම නැවත දැනුම් දෙන තුරු පමා කරන ලෙස දන්වා ඇති අතර ශ්‍රී ලංකාව බලා පිටත්ව යන පිරිස වෙත ද මෙම දැනුම්දීම් වලංගු වන බව තානාපති සඳහන් කරයි. ඊශ්‍රායලයේ සියලූ කාර්යාල හා වැඩබිම්වල කටයුතු නවතා ඇති අතර රට පුරා හදිසි තත්ත්වයක් ප්‍රකාශයට පත් කොට තිබේ.

ඊශ්‍රායලයේ සිටින ශ්‍රී ලාංකිකයන්ට තම ආරක්ෂාව සලසා ගන්නා ලෙස උපදෙස් ලබාදී ඇති අතර, කිසියම් පාර්ශවයන්ගෙන් එල්ල විය හැකි මිසයිලවලින් ආරක්ෂාව සඳහා ආරක්ෂිත කුටි ආසන්නයේ රැදී සිටින ලෙස  ඊශ්‍රායලය රජය නිවේදනය කර ඇති බව වාර්තා වේ.

එරට ප්‍රවාහන කටයුතු නවතා ඇති අතර සියලූ දෙනාට ආරක්ෂක ස්ථානවලට යන ලෙස ද උපදෙස් ලැබී තිබේ. හදිසි අවස්ථාවක දී ගිලන්රථ සේවා ක්‍රියාත්මක අතර තානාපති කාර්යාලයේ නිලධාරීන් පය 24 පුරා දුරකතනයෙන් සම්බන්ද කර ගත හැකි බව තානාපතිතුමන් විසින් සඳහන් කරයි.

සෑම අවස්ථාවකදීම බීමට අවශ්‍ය ජලය හා වියලි  ආහාර ළඟ තබා ගන්නා ලෙසත්, ඖෂධ භාවිතා කරන පිරිස නිරන්තරයෙන් ඒවා තමාගේ ආසන්න ස්ථානයක තබා ගන්නා ලෙසත් තානාපතිතුමන් විසින් වැඩිදුරටත් දන්වා සිටියි.

## ත්‍රීපෝෂ සමාගමට නව සභාපතිවරයෙක්

Published on June 13, 2025, 02:35 PM

ශ්‍රී ලංකා ත්‍රීපෝෂ සමාගමේ නව සභාපතිවරයා ලෙස දේශීය හා ජාත්‍යන්තර ව්‍යාපාරික ක්ෂේත්‍රයේ ඉහළ පළපුරුද්ද සහ අත්දැකීම් සහිත අමල් නිරෝෂණ අත්තනායක මහතා පත් කිරීමට සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍යාංශය පියවර ගෙන ඇත.

ඊට අදාල පත්වීම් ලිපිය  සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍ය වෛද්‍ය නලින්ද ජයතිස්ස මහතා විසින් අමල් නිරෝෂණ අත්තනායක මහතා වෙත ලබාදීම අද (13) සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍යාංශයේ දී සිදු විය.

වසර 25 කට අධික කාලයක් පුරා දේශීය හා ජාත්‍යන්තර ව්‍යාපාරික ක්ෂේත්‍රයේ ඉහළ පළපුරුදක් සහිත අමල් නිරෝෂණ අත්තනායක මහතා මහනුවර ධර්මරාජ විද්‍යාලයේ ආදි ශිෂ්‍යයෙකි. පාසල් වියේ  දී ජනාධිපති බාල දක්ෂයෙකු ලෙස කටයුතු කර ඇති ඒ මහතා ශ්‍රී ජයවර්ධනපුර විශ්ව විද්‍යාලයෙන් ශාස්ත්‍රවේදී ගෞරව උපාධිධාරියෙකි. විශ්ව විද්‍යාල ක්‍රීඩා සංගමයේ ලේකම්වරයා ලෙස ද කටයුතු කර ඇති ඒ මහතා ටෙනිස් ක්‍රීඩාවෙන් මෙරට නියෝජන්‍ය කරමින් ජාත්‍යන්තරය තරඟාවලි සඳහා ද සහභාගී වී ඇත.

දේශීය මෙන්ම වි‌දේශීය වෙළෙද සන්නාම රැසක නිර්මාණකරු වන අත්තනායක මහතා දකුණු ආසියාවේ මෙන් ම මෙරට ජාත්‍යන්තර ප්‍රදර්ශණ කිහිපයක් මෙහෙයවීම සහ කළමණාකරණය සිදු කර ඇත. නවීන තාක්ෂණික හා ඩිජිටල්කරණය භාවිතයේ මනා පලපුරුද්දක් ද ඒ මහතා සතු වේ.

## නොරෝච්චෝලේ  තුන්වන ජනන යන්ත්‍රය දින 25 කට නවති

Published on June 13, 2025, 01:31 PM

නොරෝච්චෝල ලක්විජය බලාගාරයෙ තුන්වන ජනන යන්ත්‍රය වාර්ෂීක නඩත්තුවක් සදහා දින 25 කට 13 දින රාත්‍රියෙ සිට ජනන කටයුතු නවතා දමන බව විදුලි බල මංඩල ඉහල නිළධාරීයකු කල විමසිමේදි පැවසිය.

ලක්විජය බලාගාරයෙ පලමු, දෙවන හා තුන්වන අධියර තුනක් ඇති අතර එයින් තුන්වන ජනන යන්ත්‍රය මෙලෙස ජනන කටයුතු නවතා වාර්ෂීකව සිදුකරන නඩත්තුව සිදු කිරීමට නියමිත බව කියයි.

නොරෝච්චෝල බලාගාරයෙන් ජාතික විදුල පද්ධතියට මෙගාවෝට් 900 කට ආසන්න විදුලි ධාරීතාවයක් ජාතික පද්ධතියට ලබා දෙන අතර තුන්වන අධියර නැවති යාමත් සමග එය මෙගාවොට් 600 කට ආසන්න සිමාවකට යටත් වන බව කියයි.

කෙසේ නමුත් තුන්වන ජනන යන්ත්‍රය දින 25 ක නඩත්තුවට නවතා දැමුවද රටේ විදුලි අවශ්‍යතාවයට කිසිදු ගැටළුවක් නොවන බව විදුලි බල මංඩල ඉහල නිළධාරීන් පවසයි.

විදුලි බව මංඩල ඉහල නිළධාරීන් පවසන්නේ ලක්විජය බලාගාරයෙ ජනන යන්ත්‍ර කාණ්ඩ කඩමින් නඩත්තු කලයුතු කාල සිමාවන් වලදි එය සිදුකරන බවත් මෙම නඩත්තුවද එවැනි වාර්ෂීකව සිදුකල යුතු නඩත්තුවක් බවයි.

ඒ අනුව මෙම නඩත්තුව සදහා දේශීය ඉංජීනේරුවන්ගෙන් සැදුම් ලත් විශේෂ කණ්ඩායමක් විසින් සිදුකිරිමට නියමිත බව කියයි.

## රු. ලක්ෂ 550ක විස්කි හා කරදමුංගු තොගයක් කටුනායක ගුවන් තොටුපොලේ දමා යයි

Published on June 13, 2025, 12:57 PM

රුපියල් පන් කෝටි පනස් ලක්ෂයක් වටිනා විස්කි බෝත්ල් 673 ක් සහ ඉන්දියාවේ නිෂ්පාදිත කරදමුංගු කිලෝ ග්‍රෑම් 75 ක් , කටුනායක ගුවන් තොටුපොළ පළමු මහළේ , තීරු බදු රහිත සාප්පු සංකීර්ණ පරිශ්‍රයේ අතහැර දමා ගොස් තිබියදී 06/13 දින අලුයම ගුවන් තොටුපොළ රේගු නිලධාරීන් පිරිසක් විසින් සොයා ගෙන අත් අඩංගුවට ගෙන තිබේ .

මෙම විස්කි සහ කරදමුංගු තොගය කුමන ගුවන් යානයක පැමිණි කුමන ගුවන් මගියෙකු විසින් කටුනායක ගුවන් තොටුපොළ වෙත රැගෙන එන්නට ඇත්දැයි සොයා රේගු නිලධාරීන් විමර්ශන කටයුතු ආරම්භ කර තිබේ .

මෙහිදී මෙම විස්කි සහ කරදමුංගු තොගය දමා ගොස් තිබූ ගුවන් තොටුපොළ තීරු බදු රහිත සාප්පු සංකීර්ණයේ , සාප්පු වල සේවකයින්ගෙන් ප්‍රශ්ණ කරමින්  සහ ගුවන් තොටුපොළ ආරක්ෂක කැමරා පද්ධතිය නිරීක්ෂණය කරමින් මේවායේ හිමිකරුවන් අනාවරණය කර ගැනීමට රේගු නිලධාරීන් ප්‍රයත්න දරමින් සිටිති .

## ඊශ්‍රායලය ඉරානයට දැවැන්ත ප්‍රහාරයක් එල්ල කරයි

Published on June 13, 2025, 11:31 AM

ඉරානයේ න්‍යෂ්ටික වැඩසටහනේ "හදවත" ඉලක්ක කරගනිමින් ඉරානය වෙත දැවැන්ත ප්‍රහාර මාලාවක් ඊශ්‍රායලය විසින් එල්ල කර තිබේ. ඉරාන රාජ්‍ය මාධ්‍යවලට අනුව, මෙම ප්‍රහාරවලින් ඉරානයේ ඉස්ලාමීය විප්ලවවාදී ආරක්ෂක බලකායේ ප්‍රධානී හොසේන් සලාමි මෙන්ම න්‍යෂ්ටික විද්‍යාඥයින් පිරිසක්ද ද මිය ගොස් ඇති බවට තහවුරු කර තිබේ.

ඊශ්‍රායල ජනාධිපති බෙන්ජමින් නෙතන්යාහු පැවසුවේ මෙම ප්‍රහාර "ඊශ්‍රායලයේ පැවැත්මට ඉරාන තර්ජනය මැඩපැවැත්වීම සඳහා ඉලක්ක කරගත් මිලිටරි මෙහෙයුමක්" බවයි.

ප්‍රතිප්‍රහාර එල්ල විය හැකි බැවින් ඊශ්‍රායලයේ හදිසි තත්වයක් ප්‍රකාශයට පත් කර තිබේ.

මෙම ප්‍රහාර සඳහා එක්සත් ජනපදය  සම්බන්ධ නොවූ බවත් කිසිදු සහායක් ලබා නොදුන් බවත් එක්සත් ජනපද රාජ්‍ය ලේකම් මාර්කෝ රුබියෝ පවසා තිබේ.

ඉරානයේ සන්නද්ධ හමුදාවන්ගේ ප්‍රකාශකයෙකු පවසා ඇත්තේ එක්සත් ජනපදය සහ ඊශ්‍රායලය යන දෙකම ප්‍රහාර සඳහා "දැඩි මිලක්" ගෙවනු ඇති බවයි.

## කළුතර ප්‍රාදේශීය සභාවේ විදුලි සේවකයා ඝාතනය කර ඇත්තේ අත් වැරදීමකින්

Published on June 13, 2025, 10:58 AM

පසුගිය  7 දා අළුයම සිය නිවසේදී කළුතර සේවකයකු කපාකොටා ඝාතනය කර ඇත්තේ අත් වැරදීමකින් බවට අත්අඩංගුවට ගත් ප්‍රධාන සැකකරුගෙන් හෙලිව ඇති බව කළුතර කොට්ඨාශ අපරාධ විමර්ශන ඒකකය පවසයි.

කළුතර ප්‍රාදේශීය සභාවේ විදුලි සේවක, දෙදරු පියෙකු වූ සුගත් ප්‍රසංග මහතා පදිංචිව සිටී කළුතර පනාපිටියේ පිහිටි නිවසට එදින අළුයම තියුණු ආයුධ රැගෙන කඩාවැදුන පිරිසක් දොර කඩා දරු දෙදෙනා ඉදිරිපිටදී කපාකොටා ඒ මහතාව ඝාතනය කර බිරිඳට කඩු ප්‍රහාරයක් එල්ලකර පලාගොස් තිබුණි.

ඒ අනුව පරීක්ෂණ කල කළුතර කොට්ඨාස අපරාධ විමර්ශන ඒකකයේ උප පොලිස් පරීක්ෂක පී ආර් වීරසේන මහතා ඇතුළු නිළධාරීන්  ප්‍රදේශයේ සීසීටීවී දර්ශණ පරීක්ෂා කිරීමෙන් පසු කළුතර මෙනේරිතැන්න ප්‍රදේශයේ පදිංචි ප්‍රධාන සැකකරු අත්අඩංගුවට ගෙන තිබේ.

ඝාතනයට යොදාගත් තියුණු ආයුධ දෙකක් සහ ඝාතනය සැලසුම් කිරීම ට යොදාගත් නවීන ජංගම දුරකථනයක් සහ ඝාතනයට පැමිණි ත්‍රීරෝද රථය ද පොලිස් භරයට ගත්බව පොලිසිය කීය.

ඝාතනයට ලක්වූ පුද්ගලයාගේ නිවස ආසන්නේ පදිංචිව සිටින  මත්ද්‍රව්‍ය ජාවාරමකට සම්බන්ධ වෙනත් පුද්ගලයකු ඝාතනය කිරීමට ලද කොන්ත්‍රාත්තුවක් මත සැකකරුවන් පැමිණ ඇති අතර, එහිදී අධිකලෙස බීමත්ව සිටී පුද්ගලයකු  ඝාතනයට ලක්වූ ප්‍රාදේශීය සභාවේ සේවකයාගේ නිවස පෙන්වීම නිසා සිදුවූ අත්වැරදීමකින් ඝාතනය සිදුව ඇති බවට ප්‍රධාන සැකකරුගේ ප්‍රකාශ වලින් හෙලිව ඇති බව පොලිසිය කියයි.

සැකකරු  කළුතර මහේස්ත්‍රාත් අධිකරණයට ඉදිරිපත් කිරීමට නියමිතව තිබුණි .

## අනුරපුරේ ආපු කට්ටිය කළා වැවත් කාලා

Published on June 13, 2025, 09:51 AM

පොසොන් වන්දනාවේ අනුරාධපුරේ පැමිණි බැතිමතුන් විනාශ කර දමා තිබු කළාවැව අවට පිරිසිදු කිරීම රාජ්‍ය නිලධාරීන් රැසකගේ සහභාගිත්වයෙන් 12 දින දවස පුරා පැවැත්විණි.

ශ්‍රී ලංකා මහවැලි  අධිකාරියේ බෝතැන්න ජලාශ භාර ප්‍රධාන ඉංජිනේරු බුද්ධික මහේෂ් නවරත්න මහතාගේ ප්‍රධානත්වයෙන් මහවැලි අධිකාරියෙ ගල්නෑව සහ ඉපලෝගම ප්‍රාදේශීය සභා දේශීය ලේකම් කාර්යාල ඇතුළු රාජ්‍ය නිලධාරීන්ගේ සහභාගිත්වයෙන් මෙම ශ්‍රම දානය පැවැත්විණි.

අනුරාධපුර වන්දනාවේ පැමිණි බොහෝ දෙනෙකුට කිලෝ මීටර් හතකින් දුර වූ අලංකාර මනස්කාන්ත  වැව් බැම්ම දැඩි ලෙස අපවිත්‍ර කොට තිබිණි.

දන්සැල් වලින් රැගෙන ආ ආහාරපාන ලබා ගෙන දිරන සහ දිරන ද්‍රව්‍ය බෝතල් මෙන්ම මත්පැන් බෝතල් ද රැසක් මෙම වැව් බැම්මේ තැන තැන දමා තිබිණි ඊට අමතරව වැව් ජලයට ද අප ද්‍රව්‍ය එක්කර තිබීම විශේෂත්වයකිි.

බලන්ගොඩ රජවක විද්‍යාලයේ අද (12දා) සවස ගසක් කැඩා වැටීම් හේතුවෙන් පාසල් සිසුවකු ජීවිතක්ෂයට පත් වූ බව බළන්ගොඩ පොලිසිය පැවසීය.

දොළහ වසර පන්ති කාමරයකට මෙම ගසක් කඩා වැටී ඇති අතර ළමුන් දාහත් දෙනෙක් බලන්ගොඩ මූලික රෝහල්ට ඇතුලත් කර තිබේ. වැඩිදුර පරීක්ෂණ බලංගොඩ පොලිසිය සිදු කරයි.

## මහනුවර මහ නගර සභාව තුළ පක්ෂ විපක්ෂ කියලා දෙකොටසක් නෑ

Published on June 12, 2025, 08:57 PM

ජාතික ජන බලවේගයට මෙවර බලය හිමිවූ මහනුවර මහ නගර සභාවේ මංගල මහා සභා රැස්වීම එහි පක්ෂ විපක්ෂ මන්ත්‍රීවරුන් 46 දෙනාගේ සහභාගිත්වයෙන් එහි ප්‍රධාන සභා ශාලාවේදී පැවැත්විණි.

මහනුවර මහ නගර සභාවේ 16 වැනි පුරපතිවරයා වශයෙන් තේරී පත්වු චන්ද්‍රසිරි විජේනායක මහතාගේ ප්‍රධානත්වයෙන් රැස්වීම  ආරම්භ වූ අතර එහිදී මෙවර නගර සභාවේ බහුතරයක් වූ නවක මන්ත්‍රීවරුන් හඳුන්වා ගැනීම සිදු විය.

එහිදී බහුතර මන්ත්‍රීවරුන්ගේ අදහස වූයේ මහනුවර මහ නගර සභාවේ පක්ෂ විපක්ෂ යනුවෙන් දෙකොට්ඨාසයක් නොසිටිය යුතු බවත් නගරයේ සුභ සිද්ධිය සහ සංවර්ධනය වෙනුවෙන් සියලු මන්ත්‍රීවරුන් එකා වැඩ කටයුතු කළ යුතු බවයි.

එහිදී නියෝජ්‍ය නගරාධපති රුවන් කුමාර ගලපිටිය මහතා කතා කරමින් අවධාරණය කර සිටියේ මහනුවර නාගරික බල ප්‍රදේශයේ ඇතැම් ප්‍රදේශවලට සභාවේ මුදල් වැය කොට සංවර්ධන කටයුතු සිදුකොට ඇතත් ඇතැම් ස්ථානවල ජනතාවට මූලික පහසුකම් හෝ නොමැති බවයි. සභාවේ දූෂණය  වංචාව අඩු වැඩි වශයෙන් තිබෙන බවත් අඩුම වශයෙන් මහජනතාවගේ බදු මුදල් නාස්තිය හෝ අවම කිරීමට පියවර ගත යුතු තිබෙන බවද පැවසීය.

මහනුවර මහ නගර ස්වභාව තුළ ආණ්ඩු පක්ෂය හෝ විපක්ෂය කියා කොටස් දෙකක් නොමැති බවත් සිටින්නේ නගරාධිපතිවරයා සහ සභිකයන් පමණක් බවද අවධාරණය කළේය.

සභාවේ ජ්‍යෙෂ්ඨ ජාතික ජනබල වේගය මන්ත්‍රී මොහොමඩ් මුතාලිබ් මහතා කරුණු දක්වමින් පවසා සිටියේ මහනුවර මානගල සභාව තුළ ඉදිරියේදී සිදුකිරීමට විශේෂ කර්තව්‍ය රැසක් ඇති බවත් ජනතාව විශාල බලාපොරොත්තුවක් ඇතුව ජාතික ජන බලවේගයට මෙම නගර සභාවේ බලය හිමිකර දී ඇති බවද එහි විශ්වාසය පළුදු නොවන ආකාරයට වැඩ කටයුතු කළ යුතුව තිබෙන බවද පැවසීය.

පසුගිය අබාවාරයේදී ජනතා විමුක්ති පෙරමුණේ තමන් ඇතුළු සභිකයන් දෙදෙනකු පමණක් සිටිමින් නගර සභාවේ දූෂණය වංචාව පිටුදැකීම සඳහා විශාල මෙහෙවරක් සිදුකළ බවත් එය ඉදිරියටත් කරගෙන යාමට ජනතාව තවත් විශාල වගකීමක් පවරා ඇති බවද අවධාරණය කළේය.

මෙරට දේශපාලන සංස්කෘතිය වෙනස් කිරීම සඳහා මෙම බලය ජනතා විසින් ලබාදී ඇති බවත් ජනතාවට තවත් සමීප වී තවත් ඉදිරියට ගෙන යා යුතුව තිබෙන බවද නගර සභාව තුළ වංචා දූෂණ රැසක් සිදුවෙමින් තිබෙන බවට ජනතාව අතර මතයක් තිබෙන බැවින් එ් පිළිබඳව විශේෂ අවධානය යොමු කොට කටයුතු කළ යුතුව තිබෙන බවද පැවසීය.

පසුගිය සභා වාරයේ මන්ත්‍රීවරුන් වශයෙන් සිටියදී තමන් ඇතුළු වත්මන් නගරාධිපතිවරයා විසින් සභාව තුළ සිදුවූ රුපියල් කෝටි පහකට අධික ජල බට වංචාවක් අල්ලස් හා දූෂණ විමර්ශන ඒකකයට යොමු කළ බවත් මෙම නගර සභා ඉතිහාසයේ විශාලතම හොරකම එයද මුතාලිබ් මහතා වැඩිදුරටත් අවධාරණය කළේය.

නගරාධිපති චන්ද්‍රසිරි විජේනායක මහතා අදහස් දක්වමින් පවසා සිටියේ සභාවේ සියලු මන්ත්‍රීවරුන්ගේ අදහස් සහ යෝජනා අසා සිටි බවත් ඒ අනුව ක්‍රියා කරමින් සභාව ඉදිරියට ගෙන යාම සඳහා සියලු මන්ත්‍රීවරුන්ගේ සහයෝගය අවශ්‍ය අවධාරණය කළේය.

මහනුවර නාගරික කොමසාරිස් ඉන්දිකා කුමාරි අබේසිංහ මහත්මිය හා නාගරික ලේකම්වරිය ඇතුළු සභාවේ ඉහළ නිලධාරීන් මෙම අවස්ථාවට සහභාගි වූහ.

## එයාර් ඉන්දියා ගුවන් අනතුරින් එක් අයෙක් බේරෙයි

## පළමු දිනයේ වැඩි ගෞරව පන්දු යවන්නන්ට

ලෝක ටෙස්ට් ශූරතාවලියේ අවසන් මහා තරගයේ දෙවන දිනය සඳහා වන තරගය අද(12දා) පැවැත්වීමට නියමිතව තිබේ.

පළමු දිනය සඳහා පන්දුවට පහරදෙමින් සිටි දකුණු අප්‍රිකා කණ්ඩායම සිය පළමු ඉනිමට පන්දුවාර 22ක් අවසානයේ කඩුලු 4ක් දැවී ලකුණු 43ක් ලබා සිටියේය.

පන්දු යැවීමේදී මිචෙල් ස්ටාර්ක් ලකුණු 10ට කඩුලු 2ක් බිඳ හෙලීමට සමත් විය.

​මෙම තරගයේ කාසියේ වාසිය දිනාගත් දකුණු අප්‍රිකානු නායක ටෙම්බා බව්මාගේ ආරාධනයෙන් පළමුවෙන් පන්දුවට පහරදුන් ඔස්ට්‍රෙලියානු කණ්ඩායම පළමු ඉනිමට සියලුදෙනා දැවී ලකුණු 212ක් රැස් කළේය.

එහිදී බොවූ වෙබ්ස්ටර් ලකුණු 72ක් සහ ස්ටීවන් ස්මිත් ලකුණු 66ක් රැස් කළහ.

පන්දු යැවීමේදී කගිසෝ රබාඩා ලකුණු 51ට කඩුලු 5ක් සහ මාර්කෝ ජැන්සෙන් ලකුණු 49ට කඩුලු 3ක් බිඳ දැමීමට සමත් විය.

ඒ අනුව ඔස්ට්‍රේලියා පළමු ඉනිමට වඩා දකුණු අප්‍රිකා කණ්ඩායම තවත් ලකුණු 169ක් පිටුපසින් පසුවන්නේය.

මේ අතර ඊයේ(11දා) දිනය පන්දු යවන්නන්ට යෝග්‍ය වූ අතර, එහිදී කණ්ඩායම් දෙකේම පන්දුවයන්නන් පළමු දිනය තුළ 14ක් බිඳ හෙලීමට සමත් විය.

අස්වැසුම දීමනාව හිමි පුද්ගලයන්ගේ ගිණුම්වලට මුදල් බැර නොකර වෙනත් පුද්ගලයන්ගේ ගිණුම්වලට මුදල්  බැර කරන ලදැයි චෝදනාවට වැඩ තහනම්ව සිටි කොළොන්න ප්‍රාදේශීය ලේකම් කාර්යාලයේ අස්වැසුම සුබසාධන අංශයේ කළමනාකාර සේවා නිලධාරිනියක් 13දා සවස ඇඹිලිපිටිය පොලීසිය මගින් අත්අඩංගුවට ගනු ලැබීය.

වයස්ගත සහ  වෙනත් අස්වැසුම මුදල් හිමිකරුවන්ගේ ගිණුම්වලට මුදල් බැර නොකර වෙනත් පුද්ගලයින්ගේ ගිණුම්වලට මුදල් බැරකරන ලද බවට ලද පැමිණිල්ලක් අනුව සැකකාරිය අත්අඩංගුවට ලත් බව ද පොලිසිය පැවසුවේය. මේවනවිට පොලීසිය කළ විමර්ශන අනුව රුපියල් තිස්හත් ලක්ෂ අසුඑක් දහසක් ( 3781000.00) එලෙස වංචා කර ඇති බව ද පොලිසිය සදහන් කළේය. විමර්ශන අවසන් නැති නිසා වංචාකරන ලද මුදල මීට වඩා වැඩිවීමට ඉඩ ඇති බව ද පොලීසිය කීවේය.

ඊශ්‍රායලය හා ඉරානය ආශ්‍රිත කලාපයේ ඇති තිඛෙන නොසන්සුන්තාවයක් හේතුවෙන් ඊශ්‍රායලය, ඉරානය ඇතුළු කලාපය පුරා ගුවන් කලාප හරහා ගුවන් ගමන් නතර කොට ඇති අතර ගුවන් ගමන් මාර්ග වෙනස් කොට යුරෝපය හා ආසියාව අතර ගමන් වාර ආරම්භ කොට තිබෙන බවත්, මේ හේතුවෙන් බොහෝ ගුවන් ගමන්වල කාලය දිගුවිය හැකිබවත්, ඊශ්‍රායලයේ ශ්‍රී ලංකා තානාපති විසින් පණිවුඩයක් නිකුත් කරමින් සඳහන් කර තිබේ.

ඊශ්‍රායලයට පැමිණීමට අපේක්ශා කරන පිරිස් තම පැමිණීම නැවත දැනුම් දෙන තුරු පමා කරන ලෙස දන්වා ඇති අතර ශ්‍රී ලංකාව බලා පිටත්ව යන පිරිස වෙත ද මෙම දැනුම්දීම් වලංගු වන බව තානාපති සඳහන් කරයි. ඊශ්‍රායලයේ සියලූ කාර්යාල හා වැඩබිම්වල කටයුතු නවතා ඇති අතර රට පුරා හදිසි තත්ත්වයක් ප්‍රකාශයට පත් කොට තිබේ.

ඊශ්‍රායලයේ සිටින ශ්‍රී ලාංකිකයන්ට තම ආරක්ෂාව සලසා ගන්නා ලෙස උපදෙස් ලබාදී ඇති අතර, කිසියම් පාර්ශවයන්ගෙන් එල්ල විය හැකි මිසයිලවලින් ආරක්ෂාව සඳහා ආරක්ෂිත කුටි ආසන්නයේ රැදී සිටින ලෙස  ඊශ්‍රායලය රජය නිවේදනය කර ඇති බව වාර්තා වේ.

එරට ප්‍රවාහන කටයුතු නවතා ඇති අතර සියලූ දෙනාට ආරක්ෂක ස්ථානවලට යන ලෙස ද උපදෙස් ලැබී තිබේ. හදිසි අවස්ථාවක දී ගිලන්රථ සේවා ක්‍රියාත්මක අතර තානාපති කාර්යාලයේ නිලධාරීන් පය 24 පුරා දුරකතනයෙන් සම්බන්ද කර ගත හැකි බව තානාපතිතුමන් විසින් සඳහන් කරයි.

සෑම අවස්ථාවකදීම බීමට අවශ්‍ය ජලය හා වියලි  ආහාර ළඟ තබා ගන්නා ලෙසත්, ඖෂධ භාවිතා කරන පිරිස නිරන්තරයෙන් ඒවා තමාගේ ආසන්න ස්ථානයක තබා ගන්නා ලෙසත් තානාපතිතුමන් විසින් වැඩිදුරටත් දන්වා සිටියි.

## බෝම්බ බියක් හේතුවෙන් එයාර් ඉන්දියා ගුවන් යානයක් හදිසි ගොඩබෑමක් කරයි

Published on June 13, 2025, 03:19 PM

තායිලන්තයේ ෆුකෙට් සිට ඉන්දියාවේ අගනුවර වන නවදිල්ලි බලා පියාසර කරමින් තිබූ එයාර් ඉන්දියා ගුවන් යානයක් බෝම්බ තර්ජනයක් හේතුවෙන් ගුවන් ගතව මද වේලාවකට පසුව යළිත ෆුකේට් ගුවන් තොටුපොළ වෙතටම ගොඩ බස්සවා තිබේ.

හදිසි සැලසුම්වලට අනුකූලව AI 379 දරණ ගුවන් යානයේ සිටි මගීන් 156 දෙනා ගුවන් යානයෙන් ඉවත් කර ඇති බව තායිලන්ත ගුවන් තොටුපළ නිලධාරියෙක් පැවසීය.

අහමදාබාද්හිදී එයාර් ඉන්දියා ගුවන් යානයක් අනතුරට පත්වීමෙන් පසුව මෙම ගුවන් සමාගමට ඇතිවූ දෙවැනි ගැටලුව මෙය වෙයි.පසුගිය වසරේ ඉන්දියානු ගුවන් සේවා සහ ගුවන්තොටුපළ ව්‍යාජ බෝම්බ තර්ජනවලින් පිරී ගිය අතර, පළමු මාස ​​10 තුළ එවැනි ව්‍යාජ ඇමතුම් හා  සහ පණිවිඩ 1,000 කට ආසන්න ප්‍රමාණයක් ලැබී තිබුණි.

## ත්‍රීපෝෂ සමාගමට නව සභාපතිවරයෙක්

Published on June 13, 2025, 02:35 PM

ශ්‍රී ලංකා ත්‍රීපෝෂ සමාගමේ නව සභාපතිවරයා ලෙස දේශීය හා ජාත්‍යන්තර ව්‍යාපාරික ක්ෂේත්‍රයේ ඉහළ පළපුරුද්ද සහ අත්දැකීම් සහිත අමල් නිරෝෂණ අත්තනායක මහතා පත් කිරීමට සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍යාංශය පියවර ගෙන ඇත.

ඊට අදාල පත්වීම් ලිපිය  සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍ය වෛද්‍ය නලින්ද ජයතිස්ස මහතා විසින් අමල් නිරෝෂණ අත්තනායක මහතා වෙත ලබාදීම අද (13) සෞඛ්‍ය හා ජනමාධ්‍ය අමාත්‍යාංශයේ දී සිදු විය.

වසර 25 කට අධික කාලයක් පුරා දේශීය හා ජාත්‍යන්තර ව්‍යාපාරික ක්ෂේත්‍රයේ ඉහළ පළපුරුදක් සහිත අමල් නිරෝෂණ අත්තනායක මහතා මහනුවර ධර්මරාජ විද්‍යාලයේ ආදි ශිෂ්‍යයෙකි. පාසල් වියේ  දී ජනාධිපති බාල දක්ෂයෙකු ලෙස කටයුතු කර ඇති ඒ මහතා ශ්‍රී ජයවර්ධනපුර විශ්ව විද්‍යාලයෙන් ශාස්ත්‍රවේදී ගෞරව උපාධිධාරියෙකි. විශ්ව විද්‍යාල ක්‍රීඩා සංගමයේ ලේකම්වරයා ලෙස ද කටයුතු කර ඇති ඒ මහතා ටෙනිස් ක්‍රීඩාවෙන් මෙරට නියෝජන්‍ය කරමින් ජාත්‍යන්තරය තරඟාවලි සඳහා ද සහභාගී වී ඇත.

## නොරෝච්චෝලේ  තුන්වන ජනන යන්ත්‍රය දින 25 කට නවති

Published on June 13, 2025, 01:31 PM

නොරෝච්චෝල ලක්විජය බලාගාරයෙ තුන්වන ජනන යන්ත්‍රය වාර්ෂීක නඩත්තුවක් සදහා දින 25 කට 13 දින රාත්‍රියෙ සිට ජනන කටයුතු නවතා දමන බව විදුලි බල මංඩල ඉහල නිළධාරීයකු කල විමසිමේදි පැවසිය.

ලක්විජය බලාගාරයෙ පලමු, දෙවන හා තුන්වන අධියර තුනක් ඇති අතර එයින් තුන්වන ජනන යන්ත්‍රය මෙලෙස ජනන කටයුතු නවතා වාර්ෂීකව සිදුකරන නඩත්තුව සිදු කිරීමට නියමිත බව කියයි.

නොරෝච්චෝල බලාගාරයෙන් ජාතික විදුල පද්ධතියට මෙගාවෝට් 900 කට ආසන්න විදුලි ධාරීතාවයක් ජාතික පද්ධතියට ලබා දෙන අතර තුන්වන අධියර නැවති යාමත් සමග එය මෙගාවොට් 600 කට ආසන්න සිමාවකට යටත් වන බව කියයි.

කෙසේ නමුත් තුන්වන ජනන යන්ත්‍රය දින 25 ක නඩත්තුවට නවතා දැමුවද රටේ විදුලි අවශ්‍යතාවයට කිසිදු ගැටළුවක් නොවන බව විදුලි බල මංඩල ඉහල නිළධාරීන් පවසයි.

විදුලි බව මංඩල ඉහල නිළධාරීන් පවසන්නේ ලක්විජය බලාගාරයෙ ජනන යන්ත්‍ර කාණ්ඩ කඩමින් නඩත්තු කලයුතු කාල සිමාවන් වලදි එය සිදුකරන බවත් මෙම නඩත්තුවද එවැනි වාර්ෂීකව සිදුකල යුතු නඩත්තුවක් බවයි.

## ඊශ්‍රායලය ඉරානයට දැවැන්ත ප්‍රහාරයක් එල්ල කරයි

ඉරානයේ න්‍යෂ්ටික වැඩසටහනේ "හදවත" ඉලක්ක කරගනිමින් ඉරානය වෙත දැවැන්ත ප්‍රහාර මාලාවක් ඊශ්‍රායලය විසින් එල්ල කර තිබේ. ඉරාන රාජ්‍ය මාධ්‍යවලට අනුව, මෙම ප්‍රහාරවලින් ඉරානයේ ඉස්ලාමීය විප්ලවවාදී ආරක්ෂක බලකායේ ප්‍රධානී හොසේන් සලාමි මෙන්ම න්‍යෂ්ටික විද්‍යාඥයින් පිරිසක්ද ද මිය ගොස් ඇති බවට තහවුරු කර තිබේ.

ඊශ්‍රායල ජනාධිපති බෙන්ජමින් නෙතන්යාහු පැවසුවේ මෙම ප්‍රහාර "ඊශ්‍රායලයේ පැවැත්මට ඉරාන තර්ජනය මැඩපැවැත්වීම සඳහා ඉලක්ක කරගත් මිලිටරි මෙහෙයුමක්" බවයි.

Published on June 13, 2025, 10:58 AM

පසුගිය  7 දා අළුයම සිය නිවසේදී කළුතර සේවකයකු කපාකොටා ඝාතනය කර ඇත්තේ අත් වැරදීමකින් බවට අත්අඩංගුවට ගත් ප්‍රධාන සැකකරුගෙන් හෙලිව ඇති බව කළුතර කොට්ඨාශ අපරාධ විමර්ශන ඒකකය පවසයි.

කළුතර ප්‍රාදේශීය සභාවේ විදුලි සේවක, දෙදරු පියෙකු වූ සුගත් ප්‍රසංග මහතා පදිංචිව සිටී කළුතර පනාපිටියේ පිහිටි නිවසට එදින අළුයම තියුණු ආයුධ රැගෙන කඩාවැදුන පිරිසක් දොර කඩා දරු දෙදෙනා ඉදිරිපිටදී කපාකොටා ඒ මහතාව ඝාතනය කර බිරිඳට කඩු ප්‍රහාරයක් එල්ලකර පලාගොස් තිබුණි.

ඒ අනුව පරීක්ෂණ කල කළුතර කොට්ඨාස අපරාධ විමර්ශන ඒකකයේ උප පොලිස් පරීක්ෂක පී ආර් වීරසේන මහතා ඇතුළු නිළධාරීන්  ප්‍රදේශයේ සීසීටීවී දර්ශණ පරීක්ෂා කිරීමෙන් පසු කළුතර මෙනේරිතැන්න ප්‍රදේශයේ පදිංචි ප්‍රධාන සැකකරු අත්අඩංගුවට ගෙන තිබේ.

ඝාතනයට යොදාගත් තියුණු ආයුධ දෙකක් සහ ඝාතනය සැලසුම් කිරීම ට යොදාගත් නවීන ජංගම දුරකථනයක් සහ ඝාතනයට පැමිණි ත්‍රීරෝද රථය ද පොලිස් භරයට ගත්බව පොලිසිය කීය.

ඝාතනයට ලක්වූ පුද්ගලයාගේ නිවස ආසන්නේ පදිංචිව සිටින  මත්ද්‍රව්‍ය ජාවාරමකට සම්බන්ධ වෙනත් පුද්ගලයකු ඝාතනය කිරීමට ලද කොන්ත්‍රාත්තුවක් මත සැකකරුවන් පැමිණ ඇති අතර, එහිදී අධිකලෙස බීමත්ව සිටී පුද්ගලයකු  ඝාතනයට ලක්වූ ප්‍රාදේශීය සභාවේ සේවකයාගේ නිවස පෙන්වීම නිසා සිදුවූ අත්වැරදීමකින් ඝාතනය සිදුව ඇති බවට ප්‍රධාන සැකකරුගේ ප්‍රකාශ වලින් හෙලිව ඇති බව පොලිසිය කියයි.

## අනුරපුරේ ආපු කට්ටිය කළා වැවත් කාලා

Published on June 13, 2025, 09:51 AM

පොසොන් වන්දනාවේ අනුරාධපුරේ පැමිණි බැතිමතුන් විනාශ කර දමා තිබු කළාවැව අවට පිරිසිදු කිරීම රාජ්‍ය නිලධාරීන් රැසකගේ සහභාගිත්වයෙන් 12 දින දවස පුරා පැවැත්විණි.

ශ්‍රී ලංකා මහවැලි  අධිකාරියේ බෝතැන්න ජලාශ භාර ප්‍රධාන ඉංජිනේරු බුද්ධික මහේෂ් නවරත්න මහතාගේ ප්‍රධානත්වයෙන් මහවැලි අධිකාරියෙ ගල්නෑව සහ ඉපලෝගම ප්‍රාදේශීය සභා දේශීය ලේකම් කාර්යාල ඇතුළු රාජ්‍ය නිලධාරීන්ගේ සහභාගිත්වයෙන් මෙම ශ්‍රම දානය පැවැත්විණි.

බලන්ගොඩ රජවක විද්‍යාලයේ අද (12දා) සවස ගසක් කැඩා වැටීම් හේතුවෙන් පාසල් සිසුවකු ජීවිතක්ෂයට පත් වූ බව බළන්ගොඩ පොලිසිය පැවසීය.

දොළහ වසර පන්ති කාමරයකට මෙම ගසක් කඩා වැටී ඇති අතර ළමුන් දාහත් දෙනෙක් බලන්ගොඩ මූලික රෝහල්ට ඇතුලත් කර තිබේ. වැඩිදුර පරීක්ෂණ බලංගොඩ පොලිසිය සිදු කරයි.

## මහනුවර මහ නගර සභාව තුළ පක්ෂ විපක්ෂ කියලා දෙකොටසක් නෑ

Published on June 12, 2025, 08:57 PM

ජාතික ජන බලවේගයට මෙවර බලය හිමිවූ මහනුවර මහ නගර සභාවේ මංගල මහා සභා රැස්වීම එහි පක්ෂ විපක්ෂ මන්ත්‍රීවරුන් 46 දෙනාගේ සහභාගිත්වයෙන් එහි ප්‍රධාන සභා ශාලාවේදී පැවැත්විණි.

මහනුවර මහ නගර සභාවේ 16 වැනි පුරපතිවරයා වශයෙන් තේරී පත්වු චන්ද්‍රසිරි විජේනායක මහතාගේ ප්‍රධානත්වයෙන් රැස්වීම  ආරම්භ වූ අතර එහිදී මෙවර නගර සභාවේ බහුතරයක් වූ නවක මන්ත්‍රීවරුන් හඳුන්වා ගැනීම සිදු විය.

එහිදී බහුතර මන්ත්‍රීවරුන්ගේ අදහස වූයේ මහනුවර මහ නගර සභාවේ පක්ෂ විපක්ෂ යනුවෙන් දෙකොට්ඨාසයක් නොසිටිය යුතු බවත් නගරයේ සුභ සිද්ධිය සහ සංවර්ධනය වෙනුවෙන් සියලු මන්ත්‍රීවරුන් එකා වැඩ කටයුතු කළ යුතු බවයි.

එහිදී නියෝජ්‍ය නගරාධපති රුවන් කුමාර ගලපිටිය මහතා කතා කරමින් අවධාරණය කර සිටියේ මහනුවර නාගරික බල ප්‍රදේශයේ ඇතැම් ප්‍රදේශවලට සභාවේ මුදල් වැය කොට සංවර්ධන කටයුතු සිදුකොට ඇතත් ඇතැම් ස්ථානවල ජනතාවට මූලික පහසුකම් හෝ නොමැති බවයි. සභාවේ දූෂණය  වංචාව අඩු වැඩි වශයෙන් තිබෙන බවත් අඩුම වශයෙන් මහජනතාවගේ බදු මුදල් නාස්තිය හෝ අවම කිරීමට පියවර ගත යුතු තිබෙන බවද පැවසීය.

මහනුවර මහ නගර ස්වභාව තුළ ආණ්ඩු පක්ෂය හෝ විපක්ෂය කියා කොටස් දෙකක් නොමැති බවත් සිටින්නේ නගරාධිපතිවරයා සහ සභිකයන් පමණක් බවද අවධාරණය කළේය.

සභාවේ ජ්‍යෙෂ්ඨ ජාතික ජනබල වේගය මන්ත්‍රී මොහොමඩ් මුතාලිබ් මහතා කරුණු දක්වමින් පවසා සිටියේ මහනුවර මානගල සභාව තුළ ඉදිරියේදී සිදුකිරීමට විශේෂ කර්තව්‍ය රැසක් ඇති බවත් ජනතාව විශාල බලාපොරොත්තුවක් ඇතුව ජාතික ජන බලවේගයට මෙම නගර සභාවේ බලය හිමිකර දී ඇති බවද එහි විශ්වාසය පළුදු නොවන ආකාරයට වැඩ කටයුතු කළ යුතුව තිබෙන බවද පැවසීය.

පසුගිය අබාවාරයේදී ජනතා විමුක්ති පෙරමුණේ තමන් ඇතුළු සභිකයන් දෙදෙනකු පමණක් සිටිමින් නගර සභාවේ දූෂණය වංචාව පිටුදැකීම සඳහා විශාල මෙහෙවරක් සිදුකළ බවත් එය ඉදිරියටත් කරගෙන යාමට ජනතාව තවත් විශාල වගකීමක් පවරා ඇති බවද අවධාරණය කළේය.

මෙරට දේශපාලන සංස්කෘතිය වෙනස් කිරීම සඳහා මෙම බලය ජනතා විසින් ලබාදී ඇති බවත් ජනතාවට තවත් සමීප වී තවත් ඉදිරියට ගෙන යා යුතුව තිබෙන බවද නගර සභාව තුළ වංචා දූෂණ රැසක් සිදුවෙමින් තිබෙන බවට ජනතාව අතර මතයක් තිබෙන බැවින් එ් පිළිබඳව විශේෂ අවධානය යොමු කොට කටයුතු කළ යුතුව තිබෙන බවද පැවසීය.

පසුගිය සභා වාරයේ මන්ත්‍රීවරුන් වශයෙන් සිටියදී තමන් ඇතුළු වත්මන් නගරාධිපතිවරයා විසින් සභාව තුළ සිදුවූ රුපියල් කෝටි පහකට අධික ජල බට වංචාවක් අල්ලස් හා දූෂණ විමර්ශන ඒකකයට යොමු කළ බවත් මෙම නගර සභා ඉතිහාසයේ විශාලතම හොරකම එයද මුතාලිබ් මහතා වැඩිදුරටත් අවධාරණය කළේය.

නගරාධිපති චන්ද්‍රසිරි විජේනායක මහතා අදහස් දක්වමින් පවසා සිටියේ සභාවේ සියලු මන්ත්‍රීවරුන්ගේ අදහස් සහ යෝජනා අසා සිටි බවත් ඒ අනුව ක්‍රියා කරමින් සභාව ඉදිරියට ගෙන යාම සඳහා සියලු මන්ත්‍රීවරුන්ගේ සහයෝගය අවශ්‍ය අවධාරණය කළේය.

මහනුවර නාගරික කොමසාරිස් ඉන්දිකා කුමාරි අබේසිංහ මහත්මිය හා නාගරික ලේකම්වරිය ඇතුළු සභාවේ ඉහළ නිලධාරීන් මෙම අවස්ථාවට සහභාගි වූහ.

## එයාර් ඉන්දියා ගුවන් අනතුරින් එක් අයෙක් බේරෙයි
